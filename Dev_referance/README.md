# Developer Reference

This folder is for development/history material that is useful to the server
maintainer but is not part of the short public setup documentation.

Contents:

- `LEGION_README.txt` — complete development summary from the original transcript through the current recovery steps.
- `CHANGELOG.txt` — detailed development changelog and verification notes.
- `_translation_workspace/` — config translation workspace and source/translated config copies.
- `build_logs/` — configure/build logs from targeted test builds. Future build logs should be placed here.

Normal users should start with:

- `../README.md`
- `../docs/setup_guide.md`
- `../docs/BUILD_WINDOWS.md`
- `../docs/PORTABLE_SERVER_SETUP.md`
